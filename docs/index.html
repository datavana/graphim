ImgNetMaker

ImgNetMaker converts a CSV file (with image URLs) or an image folder into image network-ready data for Gephi Lite (and similar tools) by downloading the images when a CSV is provided, pairing each image with its metadata, and adding two fields: a clean image filename and a Base64 data URL.

Yep — zoom through 10,000-images in a snap. 👩🏻‍💻✨❣️ Heads-up: performance depends on your browser’s available memory (RAM).

Open csv2image

📨 Input

CSV with at least one column: image_url. Ensure your file contains additional metadata columns (e.g., id, author, date, …)

⚙️ What ImgNetMaker does

  • Downloads each image_url (when allowed),
  • Derives a stable image_filename from a unique identifier in the URL,
  • Converts each image to a Base64 data URL (inline, text form of the image).

☑️ Output

  • Enhanced CSV with original columns: image_filename — clean filename used as the image key and image_dataurl — the data URL embedding the image (Base64)
  • /images/ folder with the downloaded image files (mirrors image_filename)

Open image2url

📨 Input

Provide an image folder whose filenames match the image_filename values in your dataset (CSV, JSON, or other).

⚙️ What ImgNetMaker does

  • Generates a Base64 data URL for each image,
  • Leaves your existing folder as-is (no downloads required).

☑️ Output

  • Enhanced CSV with original columns: image_filename and image_dataurl (Base64 data URL)
