ImgNetMaker

ImgNetMaker converts your CSV and/or image folder into network-ready data for visualising image networks with Gephi Lite. It downloads the images when a CSV is provided, pairing each image with its metadata, and adding two fields: a clean image filename and a Base64 data URL.

Yep — zoom through 10,000-images in a snap. 👩🏻‍💻✨❣️ Heads-up: performance depends on your browser’s available memory (RAM).

CSV → Images + Data URLs

📨 Input

CSV with at least one column: image_url. Ensure your file contains additional metadata columns (e.g., id, author, date, …)

⚙️ What ImgNetMaker does

  • Downloads each image_url (when allowed),
  • Derives a stable image_filename from a unique identifier in the URL,
  • Converts each image to a Base64 data URL (inline, text form of the image).

☑️ Output

  • Enhanced CSV with original columns: image_filename — clean filename used as the image key and img_dataurl — the data URL embedding the image (Base64)
  • /images/ folder with the downloaded image files (mirrors image_filename)

What’s next?

Build the bipartite network with Table2Net.

  • Include img_dataurl as an attribute on the image nodes (along with any other metadata).
  • Export as GEXF (nodes, edges, and attributes).

Open in Gephi Lite

  • Drag and drop the .gexf file into Gephi Lite.

Display images

  • Go to Appearance → Nodes → Images.
  • Set images from Attribute and choose the img_dataurl column.

Image folder → Data URLs

📨 Input

Provide an image folder whose filenames match the image_filename values in your dataset (CSV, JSON, or other).

⚙️ What ImgNetMaker does

  • Generates a Base64 data URL for each image,
  • Leaves your existing folder as-is (no downloads required).

☑️ Output

  • Enhanced CSV with original columns: image_filename and img_dataurl (Base64 data URL)

What’s next?

Join the enhanced CSV to your dataset using image_filename as the key.

Build the bipartite network with Table2Net

  • Include img_dataurl as an attribute on the image nodes (along with any other metadata).
  • Export as GEXF.

Open in Gephi Lite

  • Drag and drop the .gexf file into Gephi Lite.

Display images

  • Go to Appearance → Nodes → Images.
  • Set images from Attribute and select the img_dataurl column.
